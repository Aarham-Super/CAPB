CAPB FRONTEND LAYER

HoverShield

HoverShield is the user interaction layer of CAPB, responsible for rendering CAPTCHA systems and collecting verification tokens.

What It Does

HoverShield integrates Cloudflare Turnstile and Google reCAPTCHA v3 into a single interface.

Interaction Flow

  • User loads page
  • Turnstile widget is rendered
  • reCAPTCHA v3 runs silently
  • Tokens are stored temporarily
  • Data is sent to EdgeGuard

Design Goal

HoverShield is designed to be invisible, fast, and frictionless while still producing strong security signals.
