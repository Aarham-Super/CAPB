CAPB EDGE SECURITY

EdgeGuard

EdgeGuard is the backend decision engine of CAPB, responsible for verifying CAPTCHA tokens and producing a unified trust result.

Core Function

EdgeGuard processes incoming verification requests from the CAPB frontend system. It validates both Cloudflare Turnstile and Google reCAPTCHA v3 responses.

Processing Pipeline

  • Receive POST request from HoverShield
  • Validate Turnstile token
  • Validate reCAPTCHA score
  • Combine results into trust decision
  • Return structured JSON response

Security Model

EdgeGuard does not store any user data. All verification is stateless and processed at the edge using Cloudflare Workers.
